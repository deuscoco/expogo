# 📅 Daily Reminder App

<p align="center">
  A simple and efficient mobile application for managing daily tasks and reminders, built with React Native and Expo.
</p>

---

## 📖 Overview

**Daily Reminder App** is a mobile application designed to help users organize daily activities, manage task schedules, and stay productive through a simple reminder system. This project was developed as part of my mobile development portfolio to demonstrate proficiency in building cross-platform applications using modern technologies.

---

## ✨ Features

* Create and manage daily reminders
* Mark tasks as completed
* Edit and delete existing reminders
* User-friendly and responsive interface
* Local data persistence
* Fast and lightweight performance
* Cross-platform support (Android & iOS)

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm or yarn
* Expo Go (Android/iOS)

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/daily-reminder-app.git
```

Navigate to the project directory:

```bash
cd daily-reminder-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npx expo start
```

Open the Expo Go application and scan the QR Code displayed in the terminal or browser.
